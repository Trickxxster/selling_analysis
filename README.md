# 📊 Аналитика продаж STAZIS (май–июнь 2026)

Интерактивный дашборд для анализа товарного портфеля на основе данных о продажах.  
Приложение визуализирует выручку, прибыль, рентабельность и структуру продаж по категориям и товарам.

## 🔍 Возможности

- **KPI-панель** – общая выручка, валовая прибыль, средняя рентабельность, объём продаж.
- **Топ товаров** – ранжирование по выручке / прибыли, настраиваемый топ‑N.
- **Диаграмма Парето** – вклад каждого товара в общую выручку с накопительной кривой.
- **Анализ рентабельности** – гистограмма, ящик с усами, пузырьковая диаграмма «Выручка vs Рентабельность».
- **Структура по категориям** – круговая диаграмма и treemap.
- **Средняя цена и себестоимость** на единицу товара по категориям.
- **Полная таблица** с возможностью сортировки и фильтрации, экспорт в CSV.
- **Фильтры** по категориям и конкретным товарам в боковой панели.
